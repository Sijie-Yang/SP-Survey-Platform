## Use

**Hosted (recommended):** [sp-survey.org](https://sp-survey.org) — create an account and start.

**Self-host:**

```bash
git clone https://github.com/Sijie-Yang/SP-Survey-Platform.git
cd SP-Survey-Platform
npm install
cp .env.example .env   # add Supabase URL + anon key
npm run dev
```

See `.env.example` and [COMPATIBILITY.md](./COMPATIBILITY.md) for schema and release notes.
